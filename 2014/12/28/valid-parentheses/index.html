Valid Parentheses

经典栈应用

扫描串，遇到左括号就把它入栈，遇到右括号就从栈中弹出一个括号看是否配对，如果不配对或者此时栈空，返回False

如果扫描完成后栈中还有元素，返回False

扫描完后栈空，返回True

[python]
author = ‘shijiashuai’

class Solution:

# @return a boolean
def isValid(self, s):
    stack = []
    match = {
        ')': '(',
        ']': '[',
        '}': '{'
    }
    for ch in s:
        if ch in ['(', '[', '{']:
            stack.append(ch)
        elif not stack or match[ch] != stack.pop():
            return False
    if stack:
        return False
    return True
