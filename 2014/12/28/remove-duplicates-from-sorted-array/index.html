Remove Duplicates from Sorted Array

一开始没看懂题目，原来要修改传入的A数组然后返回修改后的A数组长度

[python]
author = ‘shijiashuai’
class Solution:

# @param a list of integers
# @return an integer
def removeDuplicates(self, A):
    num = None
    count = 0
    for n in A:
        if n != num:
            A[count] = n
            count += 1
            num = n
    return count

[/python]
